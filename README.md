Este proyecto trata de un sistema de escritorio desarrollada en C# que permite capturar, aguegar, eliminar y almacenar las materias y maestro del estudiante, segun su area tecnica. La aplicación utiliza arquitectura por capas, con validacionesm y botones funcionales.

 Lenguaje: C#

Framework: Windows Forms (.NET)

Arquitectura: Arquitectura por Capas

Base de datos: GESTION_ESCOLAR

DE(entorno de desarrollo): Visual Studio
